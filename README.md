# Best Construction Set Ever

## Metal parts

### Blueprints building

```
openscad parts/metal/layout.scad -D panel=[3,4] -o panel_3x4.dxf 
```
or
```
openscad parts/metal/layout.scad -D wheel=5 -o wheel_r5.dxf 
```
